# CodeAlpha_Student_Management_System
This project is a basic Student Management System developed using the C programming language. It is designed to manage student records in a simple and efficient way. The system allows the user to perform important operations such as adding student records, displaying all records, searching a student, updating student details, and deleting student records.

Each student record contains details like roll number, student name, course name, and marks. The project uses structures to store student-related information together in one place. It also uses file handling, which allows student records to be stored permanently in a file. This means that even after closing the program, the data remains saved and can be accessed again when the program is reopened.

The program is menu-driven, so the user can easily choose the required operation from the given options. Functions are used for each operation, which makes the program organized and easy to modify.

This project is very useful for beginners because it gives practical knowledge of how real-life record management systems work. It also helps students understand the importance of structures and file handling in C programming. Overall, this project is a good example of using C language to create a small but meaningful management system.

Key Features : 
1. Add student record
2. Display all student records
3. Search student by roll number
4. Update student details
5. Delete student record
6. Permanent data storage using files
   
Concepts Used : 
1. C programming
2. Structures
3. File handling
4. Functions
5. Menu-driven programming
6. Searching and updating records
